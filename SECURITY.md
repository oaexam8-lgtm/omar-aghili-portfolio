## 🚨 گزارش آسیب‌پذیری | Reporting a Vulnerability

اگر آسیب‌پذیری امنیتی پیدا کردید، لطفاً به صورت مسئولانه آن را گزارش دهید.

If you discover a security vulnerability, please report it responsibly.

### 📧 نحوه گزارش | How to Report

1. **ایمیل بفرستید به:**
   - o.a.exam8@gmail.com
   
2. **اطلاعات زیر را شامل شود:**
   - 📝 توضیحات مشکل
   - 🔍 مراحل بازتولید
   - 💥 تأثیر احتمالی
   - 💡 راه‌حل پیشنهادی (اختیاری)

3. **زمان پاسخ:**
   - ⏱️ تایید دریافت: ظرف 48 ساعت
   - 🔧 ارزیابی و رفع: ظرف 7 روز
   - 📢 اعلام عمومی: بعد از رفع مشکل

### ⚠️ لطفاً:

- ✅ مشکل را خصوصی نگه دارید تا زمانی که رفع شود
- ✅ به ما فرصت بدهید که مشکل را رفع کنیم
- ✅ از انتشار عمومی قبل از رفع خودداری کنید

### 🙏 تشکر:

از شما برای کمک به امن نگه داشتن این پروژه تشکر می‌کنیم!

Thank you for helping keep this project secure!

## 🔐 بهترین شیوه‌های امنیتی | Security Best Practices

### برای کاربران | For Users
- 🔄 همیشه از آخرین نسخه استفاده کنید
- 🔍 قبل از استفاده کد را بررسی کنید
- 🚫 اطلاعات حساس را در کد قرار ندهید

### برای توسعه‌دهندگان | For Developers
- 🔒 از HTTPS استفاده کنید
- 🧹 Input Validation انجام دهید
- 🔐 اطلاعات حساس را رمزنگاری کنید
- 📦 Dependencies را به‌روز نگه دارید
- 🧪 تست امنیتی انجام دهید
