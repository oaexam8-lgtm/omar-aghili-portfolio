# 🤝 مشارکت در پروژه | Contributing

از اینکه می‌خواهید در این پروژه مشارکت کنید متشکریم! 🎉

Thank you for wanting to contribute to this project! 🎉

## 📝 راهنمای مشارکت | Contribution Guidelines

### 1️⃣ Fork کردن پروژه
- روی دکمه **Fork** بالای صفحه کلیک کنید
- پروژه را روی اکانت خودتان کپی کنید

### 2️⃣ ایجاد Branch جدید
```bash
git checkout -b feature/your-feature-name
```

### 3️⃣ انجام تغییرات
- کد تمیز و خوانا بنویسید
- از CSS Variables استفاده کنید
- انیمیشن‌های روان بسازید
- Responsive بودن را چک کنید

### 4️⃣ Commit کردن
```bash
git add .
git commit -m "feat: توضیح مختصر تغییرات"
```

**انواع Commit:**
- `feat:` ویژگی جدید
- `fix:` رفع باگ
- `docs:` تغییرات مستندات
- `style:` تغییرات ظاهری
- `refactor:` بازنویسی کد
- `perf:` بهبود عملکرد
- `test:` اضافه کردن تست

### 5️⃣ Push کردن
```bash
git push origin feature/your-feature-name
```

### 6️⃣ Pull Request ایجاد کنید
- به repository اصلی بروید
- روی **New Pull Request** کلیک کنید
- توضیحات کامل بنویسید

## ✅ چک‌لیست قبل از Pull Request

- [ ] کد روی مرورگرهای مختلف تست شده
- [ ] Responsive بودن بررسی شده
- [ ] هیچ خطایی در Console وجود ندارد
- [ ] تم تاریک/روشن کار می‌کند
- [ ] دو زبانه بودن حفظ شده
- [ ] انیمیشن‌ها روان هستند

## 🐛 گزارش باگ | Bug Report

اگر باگ پیدا کردید:
1. مطمئن شوید قبلاً گزارش نشده
2. Issue جدید ایجاد کنید
3. توضیحات کامل بدهید
4. عکس یا ویدیو اضافه کنید

## 💡 پیشنهاد ویژگی | Feature Request

برای پیشنهاد ویژگی جدید:
1. Issue با عنوان `[Feature Request]` بسازید
2. توضیح دهید چرا این ویژگی مفید است
3. مثال‌هایی از استفاده بزنید
