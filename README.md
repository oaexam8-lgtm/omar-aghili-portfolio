## 📖 درباره پروژه | About

این پروژه یک وب‌سایت پورتفولیو شخصی است که با تکنولوژی‌های خالص وب (HTML، CSS، JavaScript) ساخته شده است. هدف از این پروژه ارائه یک نمونه تمیز، مدرن و حرفه‌ای برای معرفی مهارت‌ها و پروژه‌های شخصی است.

This is a personal portfolio website built with pure web technologies (HTML, CSS, JavaScript). The goal is to provide a clean, modern, and professional showcase for skills and personal projects.

### 🎯 اهداف پروژه | Project Goals

- ✅ طراحی مدرن و جذاب
- ✅ تجربه کاربری روان و سریع
- ✅ کد تمیز و قابل نگهداری
- ✅ پشتیبانی از چند زبان
- ✅ دسترسی‌پذیری بالا
- ✅ بهینه‌سازی برای موتورهای جستجو

---

## ✨ ویژگی‌ها | Features

### 🎨 طراحی و ظاهر
- 🌓 **تم تاریک و روشن** - تغییر آسان با یک کلیک + ذخیره ترجیحات
- 🎭 **انیمیشن‌های زیبا** - Typing effect، Scroll reveal، Hover effects
- 🎨 **طراحی مدرن** - استفاده از Gradients، Shadows، و CSS Variables
- 📐 **Grid Layout** - استفاده از CSS Grid و Flexbox

### 🌐 چند زبانگی
- 🇮🇷 **پشتیبانی فارسی** - با تغییر جهت RTL
- 🇬🇧 **پشتیبانی انگلیسی** - با جهت LTR
- 🔄 **تغییر آسان زبان** - با یک کلیک + ذخیره خودکار

### 📱 واکنش‌گرا
- 📱 **موبایل** - بهینه شده برای صفحه‌های کوچک
- 📱 **تبلت** - Layout مناسب برای صفحه‌های میانه
- 💻 **دسکتاپ** - تجربه کامل برای صفحه‌های بزرگ
- 🍔 **منوی موبایل** - با انیمیشن و Overlay

### ⚡ عملکرد و بهینه‌سازی
- 🚀 **سرعت بالا** - بدون Framework سنگین
- 💾 **ذخیره‌سازی محلی** - LocalStorage برای تنظیمات
- 🖼️ **Lazy Loading** - بارگذاری تصاویر در زمان نیاز
- 🔍 **SEO Optimized** - با Meta tags، Sitemap، و Robots.txt

### ♿ دسترسی‌پذیری
- 🏷️ **ARIA Labels** - برای Screen readers
- ⌨️ **Keyboard Navigation** - قابل استفاده با کیبورد
- 🎯 **Focus States** - نمایش واضح Focus
- 📢 **Live Regions** - اعلان تغییرات به کاربران

### 🛠️ قابلیت‌های دیگر
- 📝 **فرم تماس** - با Validation کامل
- 🔝 **Scroll to Top** - دکمه بازگشت به بالا
- 🎯 **Active Link Tracking** - مشخص کردن بخش فعال
- 🍞 **Toast Notifications** - پیام‌های زیبا
- 📊 **Skill Progress Bars** - نمایش سطح مهارت‌ها

## 🛠️ تکنولوژی‌ها | Technologies

### Frontend
- **HTML5** - ساختار معنایی
- **CSS3** - استایل‌دهی پیشرفته با Variables
- **JavaScript (ES6+)** - عملکردهای تعاملی

### Libraries & Tools
- **Font Awesome 6.6.0** - آیکون‌ها (از CDN)
- **Inter & Vazirmatn Fonts** - فونت‌های اختصاصی
- **Intersection Observer API** - انیمیشن‌های Scroll

### Development
- **Git** - کنترل نسخه
- **GitHub Pages** - میزبانی رایگان
